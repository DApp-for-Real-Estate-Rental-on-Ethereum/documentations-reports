\section*{Architecture Cloud AWS}

L'infrastructure est déployée sur AWS avec une architecture Cloud Native orchestrée par Kubernetes (EKS).

\begin{figure}[H]
    \centering
    \includegraphics[width=0.95\linewidth]{images/cloud_architecture.png}
    \caption{Architecture AWS : VPC, EKS, CloudFront, ALB, ECR, S3}
\end{figure}

\vspace{0.5cm}

\textbf{Composants principaux :}
\begin{itemize}
    \item \textbf{VPC 10.0.0.0/16} : Isolation réseau avec subnets publics/privés
    \item \textbf{EKS 1.30} : Cluster Kubernetes managé (Node Group t3.medium)
    \item \textbf{CloudFront} : CDN pour Frontend, API et Blockchain RPC
    \item \textbf{ALB} : Load balancers pour routage du trafic
    \item \textbf{ECR} : Registre des images Docker
    \item \textbf{S3} : Stockage des médias et uploads
    \item \textbf{ACM + Route 53} : Certificats SSL et DNS
\end{itemize}
